# 质量门

仅在准备进入或复核对应质量门时读取本文件；以可复现证据判断，不以角色声明判断。

| 质量门 | 必须确认 | 阻断项 |
| --- | --- | --- |
| G0 | 标准或发布通道的核心循环、最小范围和首发渠道获人工结论；可选能力默认关闭，仅对启用项、实际资源路径和后续指标决策点作结论。 | 核心体验或首发渠道不明；已启用商业能力无实现路径；实际使用资源的权属无路径。 |
| G1 | 核心循环可完成；干净环境可构建；实际使用资源可追溯；至少一个目标渠道可运行。 | 无验收条件、不能构建、已使用资源权属不明或所选渠道不可运行。 |
| G2 | 范围冻结；设备与性能证据齐备；P0/P1 已处理或获豁免；渠道差异有结论。 | 指标为空、阻断缺陷未处置、候选包不可复现。 |
| G3 | 每渠道候选包和提审资料完整；风险、回滚方案和人工放行齐备。 | 无人工放行、隐私或授权未确认、凭据管理不安全。 |

快速通道不要求进入 G0 至 G3。需求修改影响已通过质量门时，将该门标为待复核，只重跑受影响的验证项。
